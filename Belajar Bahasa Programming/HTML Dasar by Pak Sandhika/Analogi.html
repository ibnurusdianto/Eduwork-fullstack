1. HTML (Hypertext Markup Language)
        2. HTML merupakan bahasa markup bukan bahasa pemograman.
        3. Markup = Tags (Berfungsi untuk memberi tahu browser untuk menampilkan konten yang diinginkan).
        4. Orang yang menemukan HTML (Tim Bernes Lee).
        5.  HTML itu diciptakana untuk membuat struktur halaman dan menyajikan konten.
        6. CSS itu untuk mempercantikan halaman agar lebih bagus dan menarik.
        7. Java Script memiliki fungsi untuk berbagai hal, salah satunya untuk membuat website menjadi interaktif.
        8. Analogi dari HTML/CSS/Javascript seperti halnya Mankein yang memiliki kepala, tangan, dll itu merupakan bentuk manifestasi dari HTML, sedangkan CSS sendiri seperti halnya Manekin yang dikenakan baju/jas maupun topi yang berwarna-warni (colorfull), sedangkan Javascript sendiri merupakan otak atau ide dari Manekin itu sendiri sehingga bisa berjalan, bergaya dan lebih interaktif dll.
